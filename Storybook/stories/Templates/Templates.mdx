# Templates

## Configuring Storybook controls to mimic back-end custom properties
- For example, see the [choice custom properties story](/story/templates-choices-t-choice--choice-custom-properties).
- The `<script>app.RegisterProperties(...)</script>` markup contains a template literal that matches the equivalent stringified JSON custom properties generated by the back-end, with properties/values mapped to Storybook arguments.
- This allows Storybook controls to modify the custom properties, therefore updating the UI.

### Generate JSON output for ZeroHeight docs
- Uses the `parseCustomProps()` function in `./Storybook/stories/_parseCustomPropsJSON.js` to sanitise and render custom properties JSON, which can be copy/pasted by back-end users.
