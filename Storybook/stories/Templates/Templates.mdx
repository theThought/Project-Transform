# Templates
- Template HTML is copied from [project's staging server](https://online-stg.ipsosinteractive.com/mrIWeb/mrIWeb.dll?I.Project=S20223280&i.test=1).
- Uses **source HTML**, prior to any JavaScript transformations caused by `custom properties` rendered in the source HTML.
- Source HTML has been edited to provide simplified examples.

## Configuring Storybook controls to mimic back-end custom properties
- For example, see the [choice custom properties story](/story/templates-choices-t-choice--choice-custom-properties).
- The `<script>app.RegisterProperties(...)</script>` markup contains a template literal that matches the equivalent stringified JSON custom properties generated by the back-end, with properties/values mapped to story arguments.
- This allows Storybook controls to modify the custom properties, therefore updating the UI.

### Generate JSON output for ZeroHeight docs
- Uses the `parseCustomProps()` function in `./Storybook/stories/_parseCustomPropsJSON.js` to sanitise and render custom properties JSON.
