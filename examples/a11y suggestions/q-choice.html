Accessibility considerations & other notes

  • Grouped inputs (checkboxes/radios) should use <fieldset><legend>.
  • Additional instructions... probably best to go immediately before question.
    • This is because the <fieldset> needs to be a CSS Flexbox container for 2-col layout.
    • The 1st direct child must be a <legend>, so it cannot be wrapped in an extra <div> to contain other content like instructions.
  • The fieldset has an aria-describedby attribute to programmatically associate it with a unique id on the instructions. This enables screen readers to announce the instructions in addition to the fieldset's legend.
  • If status message is an error:
    • Place error message before question.
    • Shift keyboard focus to the 1st error message on the page so that it's the 1st thing announced to screen readers when page is reloaded.
    • See role="alert" and role="status" on MDN.
